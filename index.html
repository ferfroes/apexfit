API Key e preferências
API Key Anthropic
Sua API Key fica salva apenas no localStorage do seu browser. Nunca é enviada a nenhum servidor além da API Anthropic.
Dados do App

Todos os dados (perfil, check-ins, plano) ficam salvos no localStorage do seu browser. Para usar em outro dispositivo, exporte e importe o JSON.

Deploy GitHub Pages
1. Crie um repositório no GitHub (ex: apexfit)
2. Faça upload do index.html na raiz
3. Vá em Settings → Pages → Source: main / root
4. Acesse via https://seuuser.github.io/apexfit

⚠ Use sempre o mesmo browser/dispositivo para manter os dados no localStorage.
